在线书签系统 —— 让用户登录进来，保存他们的书签，并基于他们的个人喜好推荐给他们可能喜欢的其他站点

建立用户身份验证机制和个性化设置
注册
跟踪用户所感兴趣的事务并向他们显示适当的内容（个性化设置）

功能：
1. 用户登录和验证用户
2. 管理密码
3. 记录用户的个人爱好
4. 个性化内容
5. 基于已有的用户信息为用户推荐他们可能感兴趣的内容

解决方案(三个需求)：
1. 需要识别每个用户，我们应该有验证他们身份的方法
2. 需要保存单个用户的书签。用户应该能够添加和删除书签
3. 需要根据对他们的了解，向用户建议他们可能感兴趣的站点

考虑可能的扩展
1. 按主题分类的一组书签
2. 书签推荐功能中的“将此URL添加到我的书签”的链接
3. 基于数据库中最流行的URL的推荐，或者基于某一特定主题的推荐
4. 一个管理界面，用以创建、管理用户和书签
5. 使推荐书签更智能化或更快的方法
6. 附加的用户输入错误检查
实践！这是最好的学习方法。
